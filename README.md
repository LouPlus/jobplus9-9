# jobplus9-9

## 环境配置

- python3.6
- pip安装requirement.txt。命令或许是`sudo pip3 install requirement.txt`.

## 测试使用方法

先运行Flask服务器，在与manage.py同级的目录下，在命令行输入：
```shell
export FLASK_APP=manage.py
export FLASK_DEBUG=1
flask run
```

> windows下把export换成set
